A grandpa-friendly web app to create your own personal AI assistant in minutes.

**No coding. No servers. No hassle.**

## Demo

Live prototype: https://mcclowin.github.io/openclaw-deploy/

## Features

- 6-step wizard flow
- Avatar picker with emoji selection
- Personality presets (Friendly, Professional, Witty, Concise)
- Mission/purpose customization
- Channel selection (Telegram, WhatsApp, iMessage)
- Account connections (Email, GitHub, Notion, Trello, etc.)
- Skills marketplace (Weather, News, Spotify, Smart Home, etc.)

## Tech Stack

- Pure HTML/CSS/JS (no framework)
- Hosted on GitHub Pages
- Mobile-first responsive design

## Development

```bash
# Just open index.html in a browser
open index.html

# Or use a local server
npx serve .
```
